# ohjelmointi4 Project
Small social plaform where you can send short messges and see other users messages

# Libraries used
- Spring Boot   https://spring.io/projects/spring-boot
- HTMX          https://htmx.org/
- SQLite        https://sqlite.org/index.html
- thymeleaf     https://www.thymeleaf.org/

# Running
To run the server 
## Linux:
    ./mvnw spring-boot:run  -> Then navigate to localhost:8080
## Windows
    NO IDEA, i guess just run the mvnw.cmd -> Then navigate to localhost:8080
