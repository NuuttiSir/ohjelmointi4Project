# Ohjeet

1. Alussa aina git pull jolloin saa uusimmat updatet
2. Vaihda uuteen branchiin komennolla ''' git switch -c "nimi" '''
3. Tee haluamasi muutokset
4. Kun valmis muutoksien kanssa, tee ''' git add . && git commit -am "viesti" '''
5. lopuksi ''' git push origin "branch nimi" '''
6. Mene githubiin repoon kohtaan "pull requests"
7. Avaa uusi pull request ja katso, että base = main ja compare = "branch name"
8. Kirjoita hauska pull request viesti ja paina create pull request
9. Lopuksi mergee branchit pull requests sivulta
